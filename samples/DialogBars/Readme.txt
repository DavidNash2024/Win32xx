DialogBars Example
==================
This program displays a modal dialog containing a slider control, a scroll bar
control and a progress bar control. These controls are attached to CWnd objects.


Features demonstrated in this example
=====================================
* Using a dialog as an application.
* Using a slider, scroll bar and progress bar control in a dialog.
* Use of AttachItem to attach a dialog control to a CWnd.
* Use of OnMessageReflect to handle notifications in the object that
   generated them.
