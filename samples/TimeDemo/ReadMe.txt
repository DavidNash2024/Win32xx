The TimeDemo Program Features
=============================
This program demonstrates the use of CTime and CTimeSpan classes.
The CTime class represents an absolute time and date. The CTimeSpan
class defines the data type for differences between two CTime values.


Features demonstrated in this example
=====================================
* Using CMainFrame to display a frame with a menu, toolbar and statusbar.
* Use of a view window to display scrollable text.
* The various ways to construct and use a CTime object.
* Using CTimeSpan to perform time based calculations.
