This set of program files Implements the CommonDialogs sample program, which is  a modestly rich SDI application, Copyright ©2015 Robert C. Tausworthe, using the Win32++ Windows interface classes, Copyright ©2005-2015 David Nash, under permissions granted therein. This program set contains all the features of a larger form-based application, including most-recently-used list, serialization of document and frame persistent data via archive files, choice of the colors of the frame theme and all features and controls, selection of the font to be used within controls, and context help that displays topics in a compiled HTML (CHM) user guide. It was developed by the author as the starting template for developing new form-based applications.
