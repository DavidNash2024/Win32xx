Welcome


This document describes the features and operation of the CommonDialogs demo program. Salient features in the program's composition include


    • Use of Win32++, which is a C++ class library that is used to build Windows applications. It provides an application programming interface to the Windows API and is similar to MFC (however, there are several differences between it and MFC). It has the added advantage of being able to run on a wide range of free compilers, including Visual Studio Express and the MinGW compiler provided with CodeBlocks and Dev-C++.
    • Saving and retrieval of the program's persistent data in a local file using the Win32++ CArchive class.
    • Use of a separate ContextHelp class to access descriptions of the program's individual features that are contained within a compiled HTML (.chm) help document, such as the one you are now reading. The class also provides an AboutBox popup dialog.
    • Use of a separate CtlColorChoice dialog class to select colors of individual control types that  populate the form. This class is derived from the CColorChoice class, which, in turn is derived from the CColorDialog class.  It further incorporates the CListBoxDlg class that allows the user to select the control or feature whose color is being changed.  All of these classes are contained herein.
    • Use of a separate CFontChoice dialog class to select a font face, size, color, and modifiers for use as directed within the program.
    • Inclusion the program's most-recently-used (MRU) file list, fonts and colors of items within the view, and window placement data in the program's persistent data.
