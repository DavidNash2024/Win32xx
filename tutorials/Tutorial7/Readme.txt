Tutorial 7
==========
This is part of a set of tutorials which demonstrate how to use Win32++.
The tutorials are explained in detail in the tutorials section of the
help documentation.

Features demonstrated in this tutorial
======================================
* How to enable the toolbar and themes in CMainFrame::OnCreate.
* How to specify the bitmap resource in the resource script for the toolbar.
* How to specify the toolbar's buttons.
* How to use a dialog to choose the pen colour.
